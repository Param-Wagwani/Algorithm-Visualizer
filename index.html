Instructions

1. Each Algorithm will have “start” as first line and “stop” as last line for the project to work.

2. Each step of the algorithm should take one line in text area

3. Use special keywords in the line for specified nodes of the flowchart

a. Conditionals: Line should contain keyword “if” followed by the condition. And every “if” line should be eventually followed by an “else” keyword line.

b. Operations should start with keywords “then”, “perform” .

c. I O lies should have keywords “read” for reading value and “ print” for printing to screen.

d. Looping should start with keyword “while” and its condition .

Follow it up with operations ,etc and when its time to loop back use keyword “until”.

Note: Every “while” should have its own “until” down the line.
